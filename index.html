    이런 점이 편해요

    • 시각적 학습 — 메모리 블록 애니메이션으로 변수의 할당을 시각화.
    • 개념적 이해 — 컨셉 트리맵으로 C언어의 개념을 맵으로 연결해서 학습.
    • 게이미피케이션 — Achievement로 동기부여 유지, 퀴즈로 출력 예측 도전.
    • 개인 노트 — 문제마다 마크다운 메모로 복습 가능.
    02

    주요 기능

    자주 쓰는 흐름을 빠르게 쓸 수 있게 모았습니다.

    01

    메모리 시각화

    변수의 메모리 할당을 블록 단위로 애니메이션 표시. 포인터와 배열의 관계를 시각적으로 이해.

    02

    스텝 디버거

    한 줄씩 코드를 실행하면서 변수 상태의 변화를 확인. 콜 스택과 출력을 동시에 표시.

    03

    Achievement 시스템

    문제를 풀면 XP를 획득. 레벨업으로 새로운 문제 잠금 해제. 컬렉션 욕구를 자극.

    04

    출력 예측 퀴즈

    코드를 보고 출력을 예측하는 빠른 퀴즈. 지식의 정착에 효과적.

    05

    컨셉 트리맵

    C언어의 개념을 체계적으로 정리. 기초부터 응용까지 단계적으로 학습.

    06

    개인 노트

    문제마다 개인 메모 첨부. 복습과 회고에 활용.
